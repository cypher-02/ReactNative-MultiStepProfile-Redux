# ReactNative Multi-Step Profile App

## Description
A React Native application built using TypeScript and Redux Toolkit that allows users to create, view, and delete user profiles through a multi-step form.

## Features
- Multi-step profile form (Basic Info → Address → Summary)
- Redux Toolkit for state management
- React Navigation (Native Stack)
- Create and delete profiles
- TypeScript throughout the project

## Tech Stack
- React Native
- TypeScript
- Redux Toolkit
- React Navigation

## Setup Instructions
```bash
npm install
npx react-native start
npx react-native run-android
